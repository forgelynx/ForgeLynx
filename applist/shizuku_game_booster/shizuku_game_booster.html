About the App

Shizuku Game Booster uses the power of Shizuku to optimize your device for gaming. It helps fix lag and includes a Do Not Disturb (DND) mode to keep you focused.

Designed to be lightweight and respectful of your privacy.

Key Features

🔧

Shizuku Integration

runs with Shizuku permissions to access system-level settings for optimization.

🎮

GFX Tools

Unlock higher FPS and smooth graphics settings for supported games.

🚀

Box Lag Fix

Optimizes system resources to reduce lag and improve gaming performance.

🌙

DND Mode

Automatically enables Do Not Disturb mode to silence notifications while gaming.

Privacy First
