Required Permissions

To deliver device configurations, Shizuku Game Booster requests the following critical permissions:

  • Shizuku Access Allows the app to interface with hidden system APIs for resolution scaling and DNS modifications.
  • Usage Access (App Usage) Required to detect when a game is launched in the foreground to apply boosts automatically.
  • Draw Over Other Apps Necessary to render the real-time FPS and performance overlay on top of active games.
  • Manage Background Apps Used to manage active background services before a gaming session begins.
