About the App

Shizuku FPS Meter uses Shizuku in the background to power a precise FPS overlay. It is designed to be lightweight and secure, collecting no user data.

Requires overlay permission to display the FPS counter over other apps.

Key Features

🔧

Shizuku Integration

Runs with Shizuku in the background to access system-level APIs required for stable FPS overlay functionality.

📊

Overlay FPS Counter

Displays a floating FPS indicator above other apps with configurable position, size, and colors.

🛡️

No Data Collection

Does not collect, store, or share any user data. No sign-up, no analytics.

🎨

Material 3 Expressive

Rounded, vibrant, and responsive UI aligned with Material 3 expressive guidelines.

Permissions

  • Display over other apps: Required to render the FPS indicator overlay.
  • Post notifications: Optional, used to show status or quick controls.
  • Background: Uses Shizuku service when enabled to access necessary system-level APIs. User must start Shizuku and grant access.
